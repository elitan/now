# AGENTS.md

- Use the GitHub CLI `gh` for GitHub interactions.
- Prefer regular `function` declarations over arrow functions.
- This repo is TypeScript-only. Do not add JavaScript source files except unavoidable tool config shims.
- Run `npm run verify` before claiming the goal is complete.
- Useful focused checks:
  - `npm run typecheck`
  - `npm run lint`
  - `npm run format:check`
  - `npm run test`
  - `npm run build`
  - `npm run test:e2e`
  - `npm run smoke:prod`
  - `npm run smoke:bun`
